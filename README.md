# Stamon2

![stamon logo](logo/logo.svg)

## 关于Stamon2

* CLimber-Rong在2023年二月份开发了简陋的第一代Stamon。在2023年7月份，CLimber-Rong决定开发第二代——Stamon2（以下简称Stamon）
* Stamon是一门面向对象的动态编程语言。
* Stamon可移植性极高，只需要实现为数不多的依赖库，Stamon就可以在你的平台上运行。
